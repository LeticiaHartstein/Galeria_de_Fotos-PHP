# Galeria de Fotos-PHP
Uma galeria de fotos feita usando HTML, CSS e PHP 
<h2>Resumo da lógica - PHP</h2>
Usei de strings para armazenar as informações tiradas das pastas (isso feito usando o scandir), uso este processo para criar os botões com o nome das pastas dentro do diretorio principal(Images/), criar as galerias que levam o nome das pastas dentro do diretório principal e para escanear cada um destes diretorios, assim repetindo o processo, armazenando o nome das imagens em cada pasta e as exibindo as imagens
